# Agro-Beacon

GPS-Enabled Agro-Beacon for Plantation Health Monitoring and Analytics

## Overview
Agro-Beacon is an IoT-based monitoring system developed using Raspberry Pi 4 for plantation health monitoring. The system measures temperature, humidity, soil moisture, pH and wind speed. GPS Location Data and Geofencing Boundary.

## Hardware
- Raspberry Pi 4
- DHT22
- Soil Moisture Sensor
- pH Sensor
- GPS Module
- Wind Speed Sensor

## Software
- Python
- ThingsBoard
- SSH
